◆デバイスドライバのインストール
================================
  インストールの詳細はヘルプファイルを参照してください。

  デバイスドライバをインストールされますと以下のファイルがインストール
  されます。

  \<Windows Directory>\system32
      Ydci.dll            ダイナミックリンクライブラリ
      wdapi1010.dll       ダイナミックリンクライブラリ

  \<Windows Directory>\system32\drivers
      windrvr6.sys        デバイスドライバ


◆バージョン履歴
================

  Ver2.10(2010/06/03)
  -------------------
  ・デジタル出力状態確認関数を追加(YdciDioOutputStatus)
  ・リレー出力状態確認関数を追加(YdciRlyOutputStatus)

  Ver2.01(2010/05/10)
  -------------------
  ・インストール後、1ヶ月経過するとオープンできなくなる不具合を修正(Ver2.00からの問題)

  Ver2.00(2010/01/26)
  -------------------
  ・ドライバとソフトウェアパックを別々に配布するように変更
  ・Windows XP x64, Windows Vista x64, Windows 7 x64に対応
